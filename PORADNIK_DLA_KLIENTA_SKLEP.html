Poradnik obsługi sklepu (dla właściciela)

Zwięzły przewodnik – tylko realnie potrzebne rzeczy do codziennej obsługi i prostych zmian wyglądu (slider, logo, stopka).

1. Co masz?

Masz sklep na WordPress + WooCommerce. Panel logowania: /wp-admin (login + hasło). Po zalogowaniu menu po lewej.

  • WooCommerce → Zamówienia
  • Produkty → Wszystkie / Dodaj nowy
  • Produkty → Kategorie
  • WooCommerce → Raporty / Analityka
  • Użytkownicy (opcjonalnie)

2. Codzienna checklista (5 minut)

  • [ ] Zamówienia: nowe?
  • [ ] Status opłaconych → Przetwarzanie
  • [ ] Produkty oczekujące (jeśli włączone dodawanie)
  • [ ] Kluczowe stany magazynowe

3. Dodawanie produktu

  1. Produkty → Dodaj nowy
  2. Tytuł (krótki, jasny)
  3. Krótki opis (sprzedażowy) – obok zdjęcia
  4. Długi opis (opcjonalny)
  5. Kategoria (min. jedna)
  6. Cena regularna / promocyjna
  7. Magazyn: SKU + ilość (>0)
  8. Zdjęcie główne + galeria
  9. Opublikuj

Ilość 0 = produkt znika z list (ukrywanie braków włączone).

4. Edycja produktu

Produkty → najedź → Edytuj → zmień → Zaktualizuj. Cena promocyjna = stara przekreślona.

5. Kategorie

Produkty → Kategorie. Nazwa widoczna dla klienta; slug zostaw automatycznie. Możesz ustawić nadrzędną dla struktury. Nie duplikuj nazw.

6. Zamówienia

WooCommerce → Zamówienia:

  • Szare – Oczekujące
  • Żółte – Wstrzymane
  • Niebieskie – Przetwarzanie
  • Zielone – Zakończone
  • Czerwone – Anulowane/nieudane

Wejdź, sprawdź dane, przygotuj wysyłkę, po wysyłce → Zakończone.

7. Stany magazynowe

Edytuj produkt → zaznacz „Zarządzaj stanem” → wpisz ilość. Masowo: zaznacz produkty → Edycja zbiorcza → Ilość.

8. Produkty użytkowników (jeśli aktywne)

Strona „Sprzedawaj” tworzy wpisy w statusie Oczekujące. Zweryfikuj → Opublikuj albo usuń.

9. Promocje / kupony

Pojedynczy produkt: Cena promocyjna. Masowe akcje: WooCommerce → Kupony → Dodaj (kod, typ, limit).

10. Obrazy – zasady

  • JPG (zdjęcia), PNG (grafika/półprzezroczystość), WebP (jeśli dostępne)
  • Szerokość ~1200px zwykle wystarczy
  • Nazwa pliku opisowa

11. Klienci i konta

Użytkownicy – nie zmieniaj ról bez potrzeby. Reset hasła klient robi sam. Usuwając konto zdecyduj o treściach.

12. Raporty

WooCommerce → Raporty / Analityka: sprzedaż według dat, bestsellery, niskie stany (raz tygodniowo).

13. Aktualizacje i bezpieczeństwo

  • Backup przed masową aktualizacją
  • Silne hasła, minimum kont admina
  • Notuj co aktualizujesz

15. Gdy coś nie działa

  1. Tryb prywatny / odśwież
  2. Wszystkie czy pojedyncze?
  3. Co ostatnio zmieniane?
  4. Zapisz godzinę + zrzut ekranu → wyślij wsparciu

16. Priorytety

  1. Ilości > 0
  2. Szybka obsługa zamówień
  3. Dobre zdjęcia
  4. Proste promocje
  5. Małe częste poprawki

17. Mini słowniczek

  • SKU – wewnętrzny kod
  • Przetwarzanie – opłacone, w realizacji
  • Zakończone – wysłane
  • Oczekujące – płatność / publikacja

18. Lista przed kampanią

  • [ ] Grafiki gotowe

  • [ ] Testowy mail otrzymany

19. Pomoc

  • Sprzedaż: szczegóły zamówienia w panelu
  • Techniczne: opis + zrzut
  • Import/eksport pierwszy raz – pytaj

20. Esencja

Codziennie: zamówienia + stany. Tygodniowo: raport. Produkt: tytuł, cena, zdjęcie, kategoria, ilość. Promocja: cena promocyjna lub kupon. Awaria: notatka + zrzut.

21. Wygląd: slider, logo, stopka

21.1 Hero Slider

Menu: Hero Slides. Każdy slajd = wpis. Pola: nagłówek (część 1 + wyróżnione słowo), discount, opis (Enter = 2 linia), przycisk (tekst + URL), gradient, checkbox „Aktywny”. Obrazek wyróżniający = grafika. Kolejność: Quick Edit → „Kolejność”. Brak aktywnych = 3 domyślne.

21.2 Logo

Wygląd → Dostosuj → Tożsamość witryny → Logo. Alt z biblioteki (lub nazwa strony). Strona główna – logo bez linku; podstrony – link do startu.

21.3 Stopka (copyright)

Wygląd → Dostosuj → Stopka. Pole tekstowe – możesz użyć {YEAR}. Przykład: © {YEAR} PreoMarket. Wszystkie prawa zastrzeżone.

21.4 Favicon

Wygląd → Dostosuj → Tożsamość witryny → Ikona witryny. 512×512 PNG/WebP. Jeśli nie ustawisz – fallback: logo_strona.png. Po zmianie odśwież w trybie prywatnym (cache).

21.5 Szybka ściąga wizualna

  • [ ] Logo → Dostosuj → Tożsamość
  • [ ] Stopka → Dostosuj → Stopka
